# WatchDogs Clone (UE5 C++)

UE5 C++ 기반의 [와치독스1] 모작 프로젝트입니다.  
핵심 목표

1. 상호작용(Interaction) 중심의 해킹을 구축하고, 입력/대상 탐색/우선순위 처리까지 확장 가능한 구조로 만듭니다
2. TPS 슈터의 기본 구조에 불릿타임(원작 : 포커스모드)를 구현해 전투의 재미를 구현합니다
3. 간단한 파쿠르를 구현해 더 자유로운 게임 플레이를 구현합니다

---

## 개요

- [근거리 상호작용(E)]과 [해킹 상호작용(Q)] 을 분리하여 설계합니다.
- 상호작용 대상은 인터페이스/컴포넌트 확장으로 시스템을 확장할 수 있게 구성합니다.
- `LogTemp` 의존을 줄이고 기능별 전용 LogCategory(LogTPS)로 디버깅/추적이 쉬운 구조를 지향합니다.

---

## 기본 기능

- 상호작용 입력 분리
    - E: 근거리 상호작용 (예: 상자 열기, 아이템 줍기)
    - Q: 해킹/원거리 상호작용 (근거리에서도 가능)
- 상호작용 인터페이스 2종 분리(근거리/해킹)
    - 공통 형태의 함수 시그니처:
        - `CanInteract(Interactor)`
        - `GetInteractText()`
        - `GetInteractRoot()`
        - `OnInteract(Interactor)`
